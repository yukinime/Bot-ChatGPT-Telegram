
## Install & Run server

Clone the project

```bash
  git clone https://github.com/yukinime/Bot-ChatGPT-Telegram.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the server

```bash
  python main.py
```
